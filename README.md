Simple CMS
==========

**The simplest way to edit Markdown files with YAML & TOML frontmatter — completely in your browser.**

[**▶️ Try it now → simplecms.hira.im**](https://simplecms.hira.im)

[![Deploy Status](https://github.com/hsinghhira/simplecms/actions/workflows/deploy.yml/badge.svg)](https://github.com/hsinghhira/simple-cms/actions/workflows/deploy.yml) ![License](https://img.shields.io/github/license/hsinghhira/simplecms?color=%236c5ce7)

## Features

- **100% client-side** – Nothing ever leaves your browser
- No sign-up, no installation, works **offline**
- Full **YAML** and **TOML** frontmatter support
- Notion-style rich text editor
- Smart field detection (dates → datetime picker, tags → array editor, etc.)
- Auto-generates `slug` from title
- Auto-save drafts to IndexedDB
- Copy or download updated Markdown instantly
- Built with **React + Vite + Bun** (blazing fast)

Perfect for writers and developers using:

Astro • Hugo • Next.js • Jekyll • Eleventy • Hexo • Gatsby • VuePress • and more!

## Tech

- React
- Tiptap (ProseMirror-based editor)
- Vite (build tool)
- Bun (package manager & runtime)
- Lucide Icons
- IndexedDB (local draft persistence)

## Usage

1. Drag & drop any `.md` file
   OR click **"Start Fresh"** for a new post
2. Edit frontmatter & content
3. Click **Download** or **Copy** when done

That’s it — no servers, no accounts, no tracking.
